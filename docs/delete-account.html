🗑️ 帳號與資料刪除

韭菜健檢 · 由 aaowobbowocc Apps 開發

最後更新:2026-06-23

1. 概述

你隨時可以要求刪除你的「韭菜健檢」帳號與所有關聯資料。 本頁說明刪除流程、哪些資料會被刪除,以及哪些資料因法律要求需要保留。

2. 如何在 App 內刪除帳號(最快)

步驟 1. 開啟韭菜健檢 App,登入你要刪除的帳號。
步驟 2. 點左上角選單 → 進入 「我的設定」。
步驟 3. 滑到底,點 「🗑️ 刪除帳號」 按鈕。
步驟 4. 確認對話框 → 輸入你的 email 確認 → 提交。
步驟 5. 帳號與資料會在 72 小時內 從我們的資料庫完全刪除。

3. 使用 Email 申請刪除(替代方式)

如果你無法登入,或想透過 email 申請刪除,請寄信至:

📧 aaowobbowocc@gmail.com

主旨:刪除韭菜健檢帳號

內容請包含:
  • 你註冊時用的 email
  • 確認你是該帳號的擁有者(例如附最後登入時間或觀察清單片段)

我們會在 7 個工作天內 處理並回覆確認信。

4. 會刪除哪些資料

資料類型刪除狀態備註
Email、密碼雜湊 ✅ 完全刪除 72 小時內從 Supabase Auth 移除
觀察清單(股票、張數、成本) ✅ 完全刪除 同時清除 RLS 加密儲存
使用者設定 / 偏好 ✅ 完全刪除 包含 UI 風格、字體大小、語言
AI 解讀快取 ✅ 完全刪除 個人 AI 解讀的快取結果
裝置資訊 / 登入 IP ✅ 完全刪除 Supabase 後台 session 紀錄一併清除

5. 因法律或服務維運需要保留的資料

以下資料可能會基於法律或反詐用途保留 最多 90 天,但會去識別化處理, 不會再連結到你的個人身分:

  • 系統錯誤記錄(僅含時間戳 + 模糊化錯誤類型)
  • 濫用偵測記錄(若你的帳號曾觸發濫用警示)

90 天後上述資料一律永久刪除。

6. 部分資料刪除(不刪帳號)

你可以在 App 內不需要刪除帳號,自行操作以下動作:

  • 移除單一觀察清單持股(點卡片 → 編輯 → 刪除)
  • 清空全部觀察清單(設定 → 清除觀察清單)
  • 重設個人設定回預設值(設定 → 重設為預設)

這些動作不需任何申請,執行後立即生效。

7. 重要提醒

  • 帳號一經刪除 無法復原。如果你只是暫時不想用,建議登出而非刪除。
  • 刪除後若想再次使用韭菜健檢,需要重新註冊新帳號。
  • 訪客模式不會建立帳號,關閉 App 後資料自動清除,無需任何刪除動作。
